# TrabalhoReposicao_GitGithub__Cleberton-Barreto-Lico
Repositório criado para trabalho de faculdade

- `git init` – Inicializa um novo repositório Git na pasta local.
- `git add .` – Adiciona todas as modificações à área de preparação (staging area).
- `git commit -m "mensagem"` – Salva as alterações no histórico do repositório.
- `git branch branch-teste` – Cria uma nova branch chamada `branch-teste`.
- `git checkout branch-teste` – Alterna para a branch `branch-teste`.
- `git merge branch-teste` – Junta as alterações da branch secundária na branch principal (`main`).
- `git push` – Envia os commits locais para o repositório remoto no GitHub.
